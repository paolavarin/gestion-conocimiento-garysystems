# Plan de Administración de la Base de Conocimiento

Este documento describe, de forma resumida, cómo se alimenta, mantiene y audita
la Base de Conocimiento de GarySystems, alineada con prácticas de CMMI y MoProSoft.

- Alimentación mediante entregables validados.
- Control de configuración de versiones y líneas base.
- Auditorías periódicas y métricas de desempeño.
